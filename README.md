######Error Handling

--> If a user enters non-numeric values, an error message appears.

-->If the denominator is 0 during division, it displays: 'Error: The Denominator cannot be zero'.

#######Code Overview

add(a, b)	: Returns the sum of two numbers

subtract(a, b):Returns the difference

multiplication(a, b): Returns the product

division(a, b): Returns the quotient (handles divide-by-zero)
